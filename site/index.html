Port of a Classic Mac OS game to the web!

Play!

About

GLFighters was David Rosen's submission to udevgames 2001, the premiere competition for independent Mac game developers. It was originally written for Mac OS 9, the last of the Classic Mac OS era.

In 2024, I ported the game to WebAssembly and blogged about it here. It's pretty amazing that a 23-year-old game can run in a browser!

The source code is available on GitHub under the MIT license.
