# GLFighters-SDL

Port of GLFighters to SDL

## Dependencies

* C++ compiler. On Fedora, `dnf install gcc-c++`
* [SDL2](https://www.libsdl.org/). On Fedora, `dnf install SDL2-devel`
* OpenGL. On Fedora, `dnf install mesa-libGL`

## Building and running

Build with make:
```
make -C port
./port/glfighters
```
